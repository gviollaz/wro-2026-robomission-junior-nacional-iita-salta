# Onboarding - Guia para nuevos miembros

## Bienvenida

Bienvenido al equipo WRO RoboMission Junior de IITA Salta!

## Pasos iniciales

1. Crear cuenta GitHub y pedir acceso al repo
2. Instalar Pybricks en el Spike Prime: https://pybricks.com/install/
3. Abrir https://code.pybricks.com y conectar el hub por Bluetooth
4. Leer las reglas del desafio: `docs/es/competition/README.md`
5. Correr el programa de test: `software/tools/test_motores.py`
6. Leer la guia de Python Pybricks: https://docs.pybricks.com

## Recursos de aprendizaje

- [Pybricks documentacion](https://docs.pybricks.com)
- [Pybricks API reference](https://docs.pybricks.com/en/latest/)
- [WRO Learn](https://wro-learn.org) - cursos gratuitos
- [Python basico](https://www.python.org/about/gettingstarted/)

## Reglas del equipo

- Leer `CONTRIBUTING.md` antes de subir codigo
- Todo codigo lleva header (ver `AI-INSTRUCTIONS.md`)
- Probar en el robot antes de commitear
- Documentar con fotos/videos los tests
- Usar funciones para organizar el codigo
